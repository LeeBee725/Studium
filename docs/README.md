## 목표

- 회원들이 공동의 목표를 위해 모임을 생성하고 관리할 수 있는 간단한 프로그램 개발

## 요구사항

### <000> 프로그램 요구사항
1. 프로그램 이용자는 "회원"이 되어야만 프로그램을 이용할 수 있다.

### <001> "회원" 요구사항
1. "회원"는 고유식별번호, 이름을 지닌다.
2. "회원"는 "모임"에 참가한다.

### <002> "모임" 요구사항
1. "모임"은 고유식별번호, 주제, 참가자를 지닌다.
2. "모임"은 "회원"가 개설한다.
3. 개설된 모든 "모임"을 확인한다.
